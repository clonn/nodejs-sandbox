# sails-sandbox

a [Sails](http://sailsjs.org) application

# install sails
```
sudo npm -g install sails
sails new sails-sandbox
cd sails-sandbox/
sails lift
```

# generate api
```
sails generate api user
```

# start website
```
sails lift
```

# restful API
```
http://localhost:1337/user/create?name=Alin
http://localhost:1337/user
```
